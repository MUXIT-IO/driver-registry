Scpi

v1.1.1 DLL instruments by muxit

Generic SCPI instrument driver — works with any SCPI-compliant device (oscilloscopes, multimeters, function generators, power supplies, spectrum analyzers) over TCP/IP (LXI, raw socket port 5025) or Serial (RS-232/USB). Provides raw query/write access, configurable timeouts, init command sequences, and auto-connect with *IDN? identity query.
